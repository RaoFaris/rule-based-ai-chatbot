# Rule-Based AI Chatbot 🤖

## Project Overview

This project is a simple rule-based AI chatbot 

The chatbot uses predefined rules and control flow to respond to user inputs.

## Features

- Handles greetings
- Responds to predefined user inputs
- Handles unknown inputs
- Supports exit commands
- Runs in a continuous interaction loop
- Uses basic Python control flow and decision-making logic

## How to Run

1. Make sure Python is installed.
2. Open a terminal in the project folder.
3. Run:

```bash
python chatbot.py
